# Brand Font Pack Sources

This directory contains the raw font files used by the dynamic font-pack download endpoint:

- `GET /api/brand/font-pack`

Required files:

- `Anek/AnekDevanagari[wdth,wght].ttf`
- `Hind/Hind-Regular.ttf`
- `Hind/Hind-Medium.ttf`
- `Hind/Hind-SemiBold.ttf`
- `Hind/Hind-Bold.ttf`
- `JetBrains_Mono/JetBrainsMono[wght].ttf`

The endpoint validates this required set before packaging the ZIP.
